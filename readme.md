## Angularizer

Angular project boilerplate

### Stack
- NPM
- Webpack
- Bower
- Angular 1.5
- ES 6 / Babel (Module Loading)
- SASS
- Tests
  - Karma (Test Runner, http://karma-runner.github.io/)
  - Mocha (Test Framework, http://mochajs.org/)
  - Chai (BDD/TSS assertion library, http://chaijs.com/)
